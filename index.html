🧠 Core AI Concepts

LLM · RAG · Context
MCP · Agents

The five pillars of Module 1 — demystified for finance professionals with no prior coding background.

🧠
Large Language Models
Neural networks trained on trillions of tokens. Transformers predict the next word given context. BloombergGPT is finance-specific — 50B params on 363B finance tokens — outperforming GPT-4 on financial NLP tasks.
Finance: DCF commentary, earnings summarisation, credit memo drafting
📚
RAG — Retrieval Augmented Generation
Grounds LLM in real documents. A retriever fetches relevant chunks from a vector database, injects them as context, then the LLM answers from actual data — not training memory. Eliminates hallucination.
Finance: Annual report Q&A, Basel IV policy retrieval, live research queries
🔍
Context Window Engineering
The LLM's working memory — up to 200K tokens in Claude. Injecting the RIGHT data matters more than prompt wording. Context strategy determines output quality for long-form financial documents.
Finance: Full annual report analysis, multi-year comparison, deal room docs
⚙️
Model Context Protocol (MCP)
Anthropic's open standard for connecting AI to external tools, databases and APIs. Like USB-C for AI — one protocol connects Claude to Bloomberg Terminal, Excel, risk databases, or trading APIs.
Finance: Live data feeds, automated report generation, agentic deal workflows
🤖
Agentic AI & Finance Bots
AI that doesn't just respond — it acts. Multi-step agents pull live data, run analysis, generate reports, and take actions. 95% of PE firms deploying agentic AI in 2026. The future of every finance function.
Finance: Earnings dashboard bots, wealth report automation, compliance agents

Finance Prompt
Starter Kit

Battle-tested prompts for your target roles — including agents, Basel IV, earning dashboards, and wealth reports. Click to copy.

🤖 Agentic AI — Finance Workflow (MCP Pattern)
You are an AI finance agent with tools: [get_stock_data(ticker, years=3)], [get_earnings(ticker, quarters=8)], [calculate_dcf(fcf_list, wacc, g)], [send_report(email, report_text)]. Task: Analyse [COMPANY] for a buy/sell recommendation. Step 1: Fetch 3yr financials. Step 2: Pull last 8 quarters earnings vs estimates. Step 3: Run DCF base/bull/bear. Step 4: Generate investment memo and send to analyst@firm.com.
📊 Automated Earning Dashboard — AI Commentary
You are a sell-side equity analyst. Given the following quarterly earnings data: [PASTE: Revenue, EBITDA, PAT, EPS actual vs estimate for 8 quarters]. Generate: 1) Beat/miss analysis for each quarter with percentage variance, 2) Key drivers of outperformance or disappointment, 3) Trend narrative (improving/deteriorating), 4) One-sentence forward guidance sentiment. Format for a Bloomberg-style earnings dashboard panel.
⚖️ Basel IV / FRM — Pillar 3 Disclosure Agent
You are an FRM-certified risk analyst at [BANK NAME]. Using the following trading book positions and risk parameters: [PASTE DATA]. Generate a Basel IV Pillar 3 disclosure that includes: 1) SA-FRTB capital charge calculation, 2) CVA risk charge under SA-CVA, 3) Capital adequacy ratios (CET1, Tier 1, Total Capital), 4) Risk-weighted asset breakdown by asset class, 5) Board-ready risk narrative. Format per Basel Committee disclosure requirements.
💎 Wealth Management Report Generator
You are a certified financial planner generating a formal Investment Policy Statement. Client profile: Age [X], Income ₹[X] LPA, Risk tolerance: [moderate/aggressive], Goals: [Goal 1 with timeline and amount], [Goal 2]. Current portfolio: [DETAILS]. Generate a complete IPS including: Risk profile rationale, Strategic asset allocation with %, Specific instrument recommendations (MF categories, bonds, equity), SIP amounts per goal, Tax optimisation strategy (ELSS, NPS, 80C), Annual review triggers. Format as a professional wealth management report.
📚 RAG Pattern — Annual Report Q&A
You are a credit analyst. Answer the following questions using ONLY information from the annual report extract below. Do not use external knowledge. For each answer, cite the page/section. Questions: 1) Debt/EBITDA for last 3 years, 2) All disclosed covenant risks, 3) Liquidity position and current ratio trend, 4) Management's stated growth strategy, 5) Top 3 risk factors. Flag any answer where the document is insufficient. Annual Report Extract: [PASTE TEXT]
